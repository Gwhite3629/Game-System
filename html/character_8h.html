uint init_char ( Character **  character)

Initialize a character from a file.

Primary function for loading character data from a save file. Characters inventory and stats are loaded into the Character struct.

Parameters
[in,out]characterPointer to a Character type pointer which contains all relevant character data
Returns
The function returns SUCCESS if there are no errors, otherwise it returns errors according to the standard.
