Searches a file for an event.

This function searches through a file starting at the current file position in the FILE stream sent. Once the event has been found its position is written to the prof_start variable and once it has been found again the position is written to prof_end. The locator variable is the event to compare against.

Parameters
[in]fdThe file to locate the event
[in,out]prof_startPosition of event in file
[in,out]prof_endSecond position of event in file
[out]locatorEvent to search for
Returns
The function returns SUCCESS if there are no errors, otherwise it returns errors according to the standard.
